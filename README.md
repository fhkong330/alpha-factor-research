# Alpha Research Framework

This project implements a modular alpha research pipeline for systematic factor discovery and signal validation.

Pipeline:

1. Universe selection (S&P500 Technology sector)
2. Market data download
3. Data preprocessing
4. Alpha factor computation (30 WorldQuant-style alphas)
5. Cross-sectional IC evaluation
6. Factor selection
7. Composite signal construction
8. Predictive regression validation

Technologies:
Python, pandas, numpy, statsmodels
